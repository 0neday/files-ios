Files
=============

Next-generation file browser for iOS.

 * Multiple-column-UI on iPad
 * System-wide Document Provider - open files in other apps
 * Go to Folder
 * Sorting options
*  Get tfp0, root, and sandbox escape for iOS 9.0 - 10.3.3
